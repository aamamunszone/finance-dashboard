# 💲 Finance Dashboard

A high-fidelity financial management platform designed for real-time tracking, data visualization, and seamless user experience. This dashboard provides a comprehensive overview of financial health through interactive analytics and optimized state management.

## 🚀 Key Features

- **Dynamic Analytics:** Interactive data visualization using Recharts for spending trends and category distribution.

- **Transaction Intelligence:** Advanced filtering system for granular tracking of income and expenses.

- **Responsive Architecture:** Modern, utility-first UI built for desktop, tablet, and mobile performance.

- **Component-Driven Design:** Reusable, modular codebase ensuring scalability and ease of maintenance.

## 🛠️ Technical Specification

- **Core:** React 19 & React Router 7

- **Styling:** Tailwind CSS 4 & Lucide Icons

- **Visualization:** Recharts (SVG-based responsive charts)

- **Build Tool:** Vite

## 📦 Installation & Development

1. **Clone & Install:** Run `npm install` to set up all dependencies.

2. **Environment:** Execute `npm run dev` to launch the Vite development server.

3. **Build:** Use `npm run build` to generate a production-ready bundle.
